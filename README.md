# StudyHub

Aplicação web desenvolvida para estudo, com o objetivo de organizar conteúdos de aprendizado como videoaulas, PDFs e anotações, além de acompanhar o progresso de estudo.

##  Objetivo
Este projeto foi criado para praticar o desenvolvimento de aplicações web utilizando Laravel, desde a configuração do ambiente até a implementação de funcionalidades essenciais.

##  Tecnologias Utilizadas
- Laravel
- PHP
- PostgreSQL
- HTML
- CSS
- JavaScript

##  Funcionalidades Desenvolvidas
- Sistema de autenticação completo (login, registro e logout)
- Estrutura inicial para organização de conteúdos de estudo
- Base para controle de progresso de aprendizado
- Integração com banco de dados PostgreSQL

##  O que foi praticado
- Configuração de ambiente Laravel do zero
- Criação e execução de migrations
- Criação de models e relacionamento com banco de dados
- Implementação de autenticação
- Organização de projeto seguindo padrão MVC

##  Status do Projeto
Projeto de aprendizado (em andamento).

##  Observações
Projeto desenvolvido para fins de estudo e prática, com funcionalidades sendo adicionadas progressivamente.
